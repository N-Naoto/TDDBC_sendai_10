### お題1の整理
品種 と サイズ を与えて、 いちご をclassで作成する。
その文字列表現を取得する。

- 文字列表現の例
    - `あまおう: S`
    - `とちおとめ: M` 
    - `もういっこ: L`
- 品種とサイズの情報
    - 品種は、 `あまおう` `とちおとめ` `もういっこ` の3種類
    - いちごのサイズは `S` `M` `L` `LL` の4種類
